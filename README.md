## Getting Started

1.clone project ini
```bash 
git clone https://github.com/christianLuis07/next-js-Online-Store.git
```
2.install dependency
```bash
npm install
# or
yarn install
```
3. menjalankan project 
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
